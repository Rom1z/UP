# UP
# Обновление программного обеспечения предприятия (УП.)

Репозиторий содержит скрипты и документацию, разработанные в ходе учебной практики по индивидуальному заданию №14 «Обновление программного обеспечения предприятия».

## Состав репозитория

- `scripts/` — служебные скрипты для автоматизации задач администрирования.
- `docs/` — руководство пользователя и регламент обслуживания.
- `configs/` — примеры конфигурационных файлов (при наличии).

## Использование скриптов

### Установка драйверов
```powershell
# Запустите от имени администратора
.\scripts\install-drivers.ps1 -DriverPath "D:\Drivers"

```powershell
