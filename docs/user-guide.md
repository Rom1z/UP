# Руководство пользователя

## 1. Вход в систему
После включения компьютера выберите свою учётную запись и введите пароль. При первом входе система потребует сменить пароль на новый.

## 2. Доступ к общим папкам
- Откройте **Проводник**.
- В адресной строке введите `\\SRV-FILE`.
- Вы увидите папки:
  - `Docs` – общие документы;
  - `Profiles` – личные папки (доступны только администраторам);
  - `Backup` – резервные копии.
- Для удобства можно создать ярлык любой папки на рабочем столе.

## 3. Подключение к VPN (для удалённой работы)
1. Нажмите **Пуск → Параметры → Сеть и Интернет → VPN**.
2. Нажмите **Добавить VPN-подключение**.
3. Заполните поля:
   - **Поставщик VPN:** Windows (встроенное)
   - **Имя подключения:** Работа VPN
   - **Имя или адрес сервера:** (укажите внешний IP вашей организации)
   - **Тип VPN:** IKEv2
4. Сохраните и нажмите **Подключиться**, введите свои учётные данные.
5. После подключения вы сможете обращаться к общим папкам так же, как из офиса.

## 4. Действия при проблемах
- Если не открывается сетевая папка, проверьте подключение к сети (должен гореть значок сети).
- Если забыли пароль, обратитесь к системному администратору.
